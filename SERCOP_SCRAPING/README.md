# SERCOP Oportunidades

Dashboard Streamlit para identificar y priorizar oportunidades de contratación pública de Ecuador mediante la API de Datos Abiertos/OCDS del SERCOP.

## Streamlit Community Cloud

- **Main file path:** `sercop_oportunidades/app.py`
- **Python:** 3.11
- **Secrets:** no requeridos
- **Dependencias:** `sercop_oportunidades/requirements.txt`

La guía completa de uso y despliegue está en [`sercop_oportunidades/README.md`](sercop_oportunidades/README.md).

## Ejecución local

Desde la raíz del repositorio:

```powershell
pip install -r sercop_oportunidades/requirements.txt
streamlit run sercop_oportunidades/app.py
```
